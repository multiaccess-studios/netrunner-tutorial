5.6 The Corp's Turn
.1 Draw Phase
.a The Corp gains their allotted clicks
.b A paid ability window occurs, in which
  • Players may use paid abilities
  • The Corp may rez non-ice cards
  • The Corp may score agendas
.c The Corp's recurring credits refill
.d The Corp's turn formally begins
.e The Corp performs their mandatory draw
.f Play proceeds to the Action Phase
.2 Action Phase
.a A paid ability window occurs, in which
  • Players may use paid abilities
  • The Corp may rez non-ice cards
  • The Corp may score agendas
.b Does the Corp have any unspent clicks?
  • Yes The Corp takes an action
  • No Skip to 5.6.2.d
.c Return to 5.6.2.a
.d Play proceeds to the Discard Phase
.3 Discard Phase
.a The Corp discards to their maximum hand size
.b A paid ability window opens, in which
  • Players may use paid abilities
  • The Corp may rez non-ice cards
.c Does the Corp have any unspent clicks?
  • Yes The Corp loses those clicks
.d The Corp's turn formally ends
.e Play proceeds to The Runner's Turn
7.2 Accessing a Card
.1 The card is accessed
.2 The Runner may use a single mid-access ability
.3 Is the accessed card an Agenda?
  • Yes The Runner must steal it
.4 The access is complete

7.5 Breaching a Server
.1 Breach the server formally begins
.2 Is the breached server Archives?
  • Yes Turn all cards in the Corp's discard pile faceup
.3 Is the brached server HQ or R&D?
  • Yes Determine the Runners random access limit
.4 Can the runner choose an access candidate?
  • Yes The Runner chooses an access candidate.
  • No Skip to 7.5.7

.6 Return to 7.5.4

6.9 Run Timing
.1 Initiation Phase
.a The Runner announces the attacked server
.b The Runner gains bad publicity credits
.c The run formally begins
.d Is there ice protecting the attacked server?
  • Yes Proceed to the Approach Ice Phase
  • No Proceed to the Movement Phase
.2 Approach Ice Phase
.a The approach begins
.b A paid ability window occurs, in which
  • Players may use paid abilities
  • The Corp may rez non-ice cards
  • The Corp can rez the approached piece of ice
.c Is the approached ice rezzed?
  • Yes Continue to the Encounter Ice Phase
  • No Proceed to the Movement Phase
.3 Encounter Ice Phase
.a The encounter begins
.b A paid ability window occurs, in which
  • Players may use paid abilities
  • Subroutines can be broken
.c Are there unbroken, unresolved subroutines?
  • Yes The Corp resolves the next such subroutine
  • No Skip to 6.9.3.e
.d Return to 6.9.3.c
.e Continue to the Movement Phase
.4 Movement Phase
.a Approached or Encountered Ice?
  • Yes Pass that ice
.b A paid ability window opens, in which
  • Players may only use paid abilities
.c The runner may choose to jack out. Did they?
  • Yes Proceed to the Run Ends Phase
  • No Continue to 6.9.4.d
.d The Runner's position moves inward, if able
.e A paid ability window opens, in which
  • Players may use paid abilities
  • The Corp may rez non-ice cards
.f Did the Runner move inward?
  • Yes Return to the Approach Ice Phase
  • No Continue to 6.9.4.g
.g The runner approaches the attacked server
.h Continue to the Success Phase
.5 Success Phase
.a The run is declared successful
.b The Runner Breaches attacked server
.c Continue to the Run Ends Phase
.6 Run Ends Phase
.a Complete any windows open when the run ended
.b The Runner loses any unspent bad publicity credits
.c Was the Success Phase reached during this run?
  • No The run becomes unsuccessful
.d The run is complete
